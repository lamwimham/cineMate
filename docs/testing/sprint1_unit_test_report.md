## 🔧 修复的 Bug

在编写测试过程中发现并修复了以下问题：

### 1. queue.py - bytes/str 转换问题

**问题**: Redis 返回 bytes 类型数据时，`get_job_status()` 和 `cancel_job()` 无法正确处理

**修复**:
```python
# 添加 helper 函数处理 bytes/str
def get_val(key: str, default=None):
    val = data.get(key) or data.get(key.encode())
    if isinstance(val, bytes):
        val = val.decode()
    return val or default
```

**影响**: 确保 Redis 响应兼容性

### 2. queue.py - 空值处理

**问题**: `cancel_job()` 在 Job 不存在时返回 True (应为 False)

**修复**:
```python
# 添加 None 检查
if current_status is None:
    return False
```

**影响**: 正确的错误处理
